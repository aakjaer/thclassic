Specialist i import-biler med lavt km-tal

Vi har specialiseret os i handel med fine import-biler med lavt km-tal, som er uden rust og slidtage. Vi beskæftiger os hovedsagelig med import af Mercedes Benz, men har også enkelte andre mærker på lager. Det vigtigste for os er, at bilerne er originale og velholdte. Vi importerer udelukkende biler, som vi selv kan lide og som vi kan stå inde for i enhver henseende. Se salgslisten...

Hvorfor vælge TH Classic?

Det er en klar strategi fra vores side at selskabet ikke skal vokse sig stort og upersonligt. For os er den personlige kontakt og opfølgning af hver enkelt handel vigtig. Hos os finder De ingen smarte sælgere eller arrogante værkførere, der forsøger at bortforklare eventuelle problemer.
Hver gang De kontakter os, vil De blive betjent af chefen selv, som er direktør Torben Stiig Hansen. Jeg kan selvstændigt tage stilling til enhver henvendelse, så kommandovejen hos os er kort og effektiv. Læs mere...
